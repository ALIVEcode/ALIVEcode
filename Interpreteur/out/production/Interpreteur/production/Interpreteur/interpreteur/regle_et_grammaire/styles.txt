"""

structure:
	nom_de_la_regle {
		nom_param_du_style : valeur
	}
	
	OU
	
	.nom_de_la_categorie {
		nom_param_du_style : valeur
	}
	

options possibles pour styliser:
	couleur -> changer la couleur (la couleur doit être donnée en RGB séparé de virgule (R,G,B))
	italique -> true ou false
	gras -> true ou false
	souligner -> true ou false
	taille -> chiffre
	fontFamily -> nom_de_la_font_family

ex:
nom_de_la_regle {

	couleur : 255,0,0
	
	italique : true

}

TIP: si tu veux appliquer le meme style à plusieurs regles/categories, sépares les noms des regles et des categories par |

ATTENTION: il est important de ne PAS enlever les styles '(defaut)' et '(erreur)'!! Cependant, vous pouvez
modifier leur contenu sans aucun problème

ATTENTION part 2: il est important que le premier style défini soit celui par défaut!!

"""
